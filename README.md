# Mini juego web de plataformas

Plantilla base inspirada en el estilo de un juego de plataformas tipo Mega Man.

## Como usarlo

1. Abre [index.html](/Users/paul/GITHUB/MEHGAMAN/index.html) en el navegador.
2. Usa flechas o `A` `D` para moverte.
3. Usa `W`, flecha arriba o espacio para saltar.
4. Usa `J` para disparar.
5. Usa `R` para reiniciar.

## Estructura

- `index.html`: contenedor de la pagina y carga de Phaser.
- `style.css`: interfaz y layout responsive.
- `game.js`: logica principal del jugador, enemigos, balas y victoria.

## Siguientes mejoras recomendadas

- Reemplazar las texturas generadas por sprites reales.
- Crear varios niveles usando Tilemaps.
- Agregar animaciones, sonidos y jefes.
- Separar la logica en varias escenas: menu, nivel, game over.
